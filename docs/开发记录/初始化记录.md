## 待手动执行的操作

网络恢复后，在项目根目录执行：

```bash
pnpm install --registry https://registry.npmmirror.com
```

如果 npmmirror 仍然不稳定，可尝试：

```bash
pnpm install --registry https://registry.npmjs.org
```

安装完成后，验证项目可正常运行：

```bash
pnpm dev
```
