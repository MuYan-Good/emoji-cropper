# 功能需求：图片压缩

## 功能概述

为表情包裁剪工具新增独立的图片压缩模块，通过顶部 Tab 导航切换，帮助用户对图片进行质量压缩和尺寸调整，满足微信表情开放平台体积限制、日常分享传输优化、存储空间节省等多场景需求。

## 用户故事

**作为** 表情包创作者/整理者
**我想要** 对图片进行压缩和尺寸调整
**以便于** 满足平台体积要求、便于分享传输、节省存储空间

## 核心功能

### 1. 导航与模块切换

- 顶部 Tab 导航栏，包含「裁剪工具」和「图片压缩」两个标签页
- Tab 切换时保持各模块状态独立，互不干扰
- 默认展示「裁剪工具」页面

### 2. 图片上传

- 支持拖拽或点击上传多张图片（批量模式）
- 支持的输入格式：PNG、JPEG、GIF
- 显示每张图片的原始信息：文件名、尺寸、体积
- 支持添加/移除单张图片

### 3. 压缩参数控制

- **质量滑块**：0-100% 范围，默认值 80%
  - 仅对 JPEG 输出有效（PNG 为无损格式，Canvas API 不支持 PNG 有损压缩）
  - 拖动滑块时实时预览压缩效果
  - 实时显示压缩后预估体积和压缩率
- **输出格式选择**：
  - PNG → PNG（保持透明背景，仅通过缩小尺寸减小体积）
  - PNG → JPEG（不支持透明背景，质量滑块有效，体积更小）
  - JPEG → JPEG（质量滑块有效）
- **尺寸调整**：
  - 支持按百分比缩放（25%、50%、75%、100%、自定义）
  - 支持锁定/解锁宽高比
  - 支持自定义宽高像素值输入

### 4. 实时预览

- 拖动质量滑块时实时更新压缩预览图（JPEG 输出时）
- 显示压缩前后对比信息：
  - 原始体积 vs 压缩后体积
  - 压缩率百分比
  - 原始尺寸 vs 输出尺寸
- 批量模式下显示总体积变化汇总

### 5. 批量处理与导出

- 统一压缩参数，一键批量压缩所有上传图片
- **单张下载**：点击单张图片可单独下载压缩结果
- **批量打包**：一键将所有压缩结果打包为 ZIP 文件下载
- 压缩过程中显示进度状态

## 边界情况

| 场景 | 预期行为 |
|------|----------|
| 上传 GIF 文件 | 显示提示"GIF 格式暂不支持压缩"，或仅作为输入展示不支持压缩操作 |
| 质量设为 0% | JPEG 输出质量极低但文件最小，允许此操作 |
| 尺寸设为 0 | 输入校验阻止，提示"尺寸必须大于 0" |
| 上传超大图片（>10MB） | 校验拦截，提示文件大小超限 |
| PNG→JPEG 时有透明区域 | 透明区域填充白色背景 |
| 批量上传 0 张图片 | 压缩按钮禁用状态 |

## 不在范围内

- GIF 格式压缩（后续迭代引入 GIF 编码器后支持）
- WebP 格式输出
- 目标体积模式（自动计算压缩参数以达到指定体积）
- 压缩参数的逐图个性化设置（批量统一参数）
- PNG 有损压缩（颜色量化，需引入额外库）
- 压缩历史记录
